# iTrustU-loopback
Si Slamet Backend using Loopback


## Install Guide
1. Clone this repo
2. ```$ cd i-trust-u-back-end```
3. ```$ cp .env.example .env```
4. Isi credential pada .env
5. Run with command : ```$ node .```


### Notes
- Environtment Variable using [dotenv](https://github.com/motdotla/dotenv)
